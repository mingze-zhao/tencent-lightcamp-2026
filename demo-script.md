# 家访秒记 P2 前端路演演示脚本

## 场景 1: 多老人列表管理与选择
**操作：** 讲者打开应用，左侧侧边栏展示当前负责跟进的多位长者列表（陈伯、李婆婆、张叔）。
**话术：** “各位评委好，这是我们专门为香港社工打造的‘家访秒记’工作台。社工一人要跟进几十位长者，所以我们首屏左侧就是按高、中、低风险排序的长者列表，一目了然。”
**操作：** 点击“陈伯”卡片。

## 场景 2: 开始家访录音与实时转写
**操作：** 点击主屏左上方“开始录音”按钮。按钮变为红色并闪烁，显示录制中时间（如 `12:45`）。
**话术：** “当我们到陈伯家进行家访时，只需把手机或平板放在桌上，点击录音。系统会自动过滤背景杂音，并开始实时语音转文字。我们的左侧面板就像一个文字编辑器一样，把老人的粤语对话进行转写，并按说话人清晰分离。”
**操作：** 鼠标上下滚动左侧 Transcript 区域，展示高亮的粤语内容。

## 场景 3: 自动化信息抽取与风险分级预警
**操作：** 引导视线到右侧面板。展示高风险预警中心。
**话术：** “最神奇的地方在右边。传统模式下社工要在回办公室后凭记忆手写报告。但在我们的系统中，当转写完成后，LLM 会立即自动抽取关键维度：如用药依从性、身体症状、情绪和社交。不仅如此，系统还会基于设定规则对隐患自动打上风险等级标签。”
**操作：** 鼠标点击或指向红色的“高风险预警 - 连续漏服降压药”。
**话术：** “例如陈伯提到自己前两天停了降压药，系统瞬间将其捕捉，并标为红色高风险预警，防止人工疏漏。”

## 场景 4: 一键生成报告与待办跟进
**操作：** 滑动右侧面板到底部的待跟进事项，然后点击右上角“生成报告”按钮（目前作为模拟效果展示）。
**话术：** “除了提取信息，系统也自动生成了规范的社福机构格式摘要，并把需要跟进的事项如‘联系门诊医生确认降压药服用方案’列为待办。从此，社工的每分钟家访时间，都能成为长者生命的保障！”
