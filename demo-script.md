# 家访秒记 P2 前端路演演示脚本（全量版）

## 场景 1: 档案二级目录 + 标签搜索
**操作：** 展示左侧长者档案，输入“高血压”或点击标签筛选。
**话术：** “左侧已支持‘姓名 + 标签’联合搜索，方便社工快速定位目标个案。”
**操作：** 点击“陈伯”，展开二级目录（采访记录）与日历。

## 场景 2: 左下固定日历入口 + 按日切换采访
**操作：** 在左下固定“采访日历入口”点击有记录的日期（带长者名标注）。
**话术：** “这个入口固定在左下，只有有采访记录的日期才会出现并高亮。点日期会直接跳到当天相关长者与会话。”

## 场景 3: 左右对称联动（短语级）
**操作：** 在左侧点击一句里的高亮短语（不是整句），观察右侧跳转。
**话术：** “每条结构化信息都绑定原文短语，支持双向跳转，保证可追溯与可核对。”
**操作：** 在右侧点击某个 Source 引用 chip，观察左侧定位到具体短语。

## 场景 4: 增量生成式右侧更新
**操作：** 左侧“Append one sentence...”输入一条新句子，点击 `Incremental`。
**话术：** “每新增一句，系统就触发一次增量抽取，右侧自动新增结构化块，并带原文引用。”

## 场景 5: 人体正反面分区图 + 时间线
**操作：** 在右侧切换人体图“正面/背面”，点击分区标签，再拖动时间线（Timeline）。
**话术：** “人体图是可点击分区的，支持前后视。每个部位按新发/持续/痊愈着色，点击后可反向定位对应原文证据。”

## 场景 6: 社区统计热力图
**操作：** 切换顶部 tab 到 `Community` 页面。
**话术：** “这里聚合所有长者采访，展示部位级热力图和共性问题排名，帮助机构做群体干预策略。”

## 场景 7: 演示模式 / 实机模式
**操作：** 打开设置，切换 `mode` 为 `demo/live`。
**话术：** “演示模式走本地 Mock；实机模式直接连 Node + Prisma + SQLite，数据重启后仍保留。” 

## 场景 8: 增量抽取 API 占位
**操作：** 切到 live 模式后，在左侧输入一句新转写并点击 `Incremental`。
**话术：** “每新增一句会先写入 transcript，再调用增量抽取接口生成右侧结构化块与 sourceRef，实现持续生长的数据流。”
